Install SecureVault Today

Install SecureVault as a Firefox extension to start protecting your credentials with zero-plaintext security.

Installation Guide

1
Open Firefox Developer Tools

Navigate to about:debugging#/runtime/this-firefox

2
Load Temporary Add-on

Click "Load Temporary Add-on" and select the manifest.json file

3
Start the Backend

Run npm start in the project root to start the CPU server on port 8080
